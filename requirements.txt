# Pygame — core 2D game framework
# Provides: window creation, 2D surface rendering, sprite blitting, event/input
# handling (keyboard + mouse), image loading (PNG, JPG, GIF, WebP), system font
# rendering (SysFont), display management, clock/tick timing, surface
# transforms (smoothscale, rotate, flip), and colour/rect utilities.
#
# Pygame APIs used in this project:
#   pygame.init / pygame.quit           — lifecycle (main.py)
#   pygame.display.set_mode / flip      — window management (app.py, menu.py)
#   pygame.display.Info                  — screen resolution query (app.py, menu.py)
#   pygame.display.set_caption           — window title (app.py)
#   pygame.event.get                     — input event loop (app.py, menu.py)
#   pygame.time.Clock.tick               — 60 FPS frame cap (app.py)
#   pygame.image.load / convert_alpha    — art asset loading (assets.py, menu.py)
#   pygame.transform.smoothscale         — tile & background scaling (assets.py, menu.py)
#   pygame.Surface / SRCALPHA            — off-screen surfaces for flash overlay (app.py)
#   pygame.draw.rect / line              — fallback tile rendering, HUD panel (app.py)
#   pygame.font.SysFont                  — HUD text rendering (app.py)
#   pygame.Rect / collidepoint           — hit-testing for menu buttons (menu.py)
#
# Homepage: https://www.pygame.org
# Licence:  LGPL v2.1
# Note:     pygame-ce (community edition, https://pyga.me) is a fully
#           compatible drop-in replacement and also satisfies this requirement.

# PLUGINS / EXTENSIONS

# None at this time. The game uses no Pygame plugins or external extensions.
# All rendering, input, and audio are handled through the base Pygame API.
#
# Pygame sub-modules used:
#   - pygame.mixer  — audio system: mixer.init, mixer.Sound, mixer.Channel
#                     Used in game/audio.py for procedurally generated SFX
#                     and looping background music (no external audio files).
#
# The game does NOT use:
#   - pygame.sprite (characters drawn manually, not via Sprite groups)
#   - pygame.joystick (keyboard/mouse only)
#   - Any external level editor or map format (levels are ASCII-defined)
#
# Additional standard library modules used by the audio system:
#   math   — sin, cos, pi for sine wave generation (audio.py)
#   struct — pack / pack_into for 16-bit PCM sample encoding (audio.py)
